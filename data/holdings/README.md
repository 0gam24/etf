# ETF 구성종목(Holdings) 캐시

각 파일명은 KRX ETF 종목코드(예: `0080G0.json`)와 일치합니다. 메인 페이지의
"오늘 거래량 1위" 카드 + ETF 상세 페이지에서 사용됩니다.

## 데이터 출처

운용사(미래에셋자산운용, 삼성자산운용, 한화자산운용 등) **PDF/엑셀 일별 공시**를
원본으로 합니다. 파이프라인이 자동 갱신되기 전까지는 **수기 캐시**(아래 스키마)로
대체합니다.

> 이 폴더의 데이터는 운용사 공시(KRX·발행사 페이지)를 기준으로 한 **대표 비중 스냅샷**입니다.
> 정확한 일자별 비중은 각 발행사 홈페이지에서 확인하세요.

## 스키마

```json
{
  "code": "0080G0",
  "name": "KODEX 방산TOP10",
  "asOf": "2026-04-22",
  "source": "samsungfund.com",
  "type": "equity",
  "holdings": [
    { "name": "한화에어로스페이스", "ticker": "012450", "weight": 24.8 }
  ]
}
```

- `type`: `equity` | `bond` | `commodity` | `mixed` — 단일자산(금현물 등)은 holdings 표시 생략
- `weight`: 퍼센트(%) 단위, 합계는 100을 넘지 않도록
- `ticker`: 한국주식 6자리 또는 US 티커. 없으면 생략 가능

## 추가/갱신 방법

1. 발행사 사이트에서 PDF/엑셀 다운
2. 상위 5~10종목과 비중을 추려 위 스키마로 작성
3. 파일명은 ETF 종목코드 그대로 (`{code}.json`)
